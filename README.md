# StarQuest
My final project for CS50P is a four‑game collection where the player competes against the computer. The bundle includes:  Tic‑Tac‑Toe (X‑O)  Rock–Paper–Scissors  Number Guessing Games Trivia Q&amp;A Game  Each game rewards the player with stars based on their performance The goal is to create a fun, interactive experience that combines classic games
